# Assets Folder

Folder ini untuk menyimpan file gambar seperti logo dan foto.

## File yang Dibutuhkan:

- `logo.png` - Logo aplikasi
- `slide1.jpg` - Gambar slide 1 untuk display
- `slide2.jpg` - Gambar slide 2 untuk display
- `slide3.jpg` - Gambar slide 3 untuk display

## Cara Upload:

Upload file gambar ke folder ini dengan nama yang sesuai.

## Slideshow Display:

- Slideshow otomatis berganti setiap 5 detik
- Tambahkan lebih banyak slide dengan menambahkan gambar dan update display.html
