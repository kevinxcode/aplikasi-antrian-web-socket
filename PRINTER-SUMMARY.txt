üöÄ CARA MENGGUNAKAN:

1. INSTALL DEPENDENCIES
   cd app
   npm install

2. HUBUNGKAN PRINTER
   - Colokkan printer thermal 58mm ke USB
   - Nyalakan printer

3. TEST PRINTER
   node test-printer.js

4. JALANKAN APLIKASI
   npm start

5. KONFIGURASI STRUK
   - Login sebagai admin
   - Klik menu "Printer"
   - Isi form pengaturan
   - Klik "Test Print"
   - Klik "Simpan Pengaturan"

6. TEST AMBIL NOMOR
   - Buka: http://localhost:3000/ambil-nomor.html
   - Klik button CS atau Teller
   - Printer otomatis cetak struk!

üêõ TROUBLESHOOTING:

PRINTER TIDAK TERDETEKSI:
- Cek koneksi USB
- Restart printer
- Coba port USB lain

ERROR LIBUSB_ERROR_ACCESS (Windows):
1. Download Zadig: https://zadig.akeo.ie/
2. Jalankan Zadig
3. Pilih printer ‚Üí Install WinUSB driver
4. Restart aplikasi

PRINT TIDAK KELUAR:
- Cek kertas thermal masih ada
- Cek printer tidak paper jam
- Test print dari button test
